# Instructions for Repository Agents

Preserve continuity.

Before changing anything, read `COVENANT.md`, `CONSTITUTION.md`, `ARCHITECTURE.md`, and `ROADMAP.md`.

Prefer small, reversible changes. Never delete working code merely because a rewrite appears cleaner.

Do not commit secrets, model weights, virtual environments, caches, or private databases.

For self-modification, require:

1. proposal
2. backup
3. patch
4. syntax check
5. tests
6. health check
7. review
8. promote or rollback
