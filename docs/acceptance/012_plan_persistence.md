# 012 — Plan Persistence

## Conversation

USER: My goal is to add vision routing.

USER: What is my current plan?

## Required

- one plan exists
- plan persists
- asking again does not create another plan
- plan contains deterministic vision-routing steps

Then:

USER: The vision model is selected.

## Required After Update

- existing plan updates
- first step completes or becomes satisfied
- next step becomes model verification
- plan ID remains unchanged
